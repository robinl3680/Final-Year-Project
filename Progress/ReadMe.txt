If you want to run the program execute the 'testbash.sh' included in the "Progress" folder.
You can edit the testbash.sh to change the input files, some example input are provided in that folder.
Three algorithms using different approaches for communication are included in the folder. 

Inorder to compare between the greedy approach and peer to approach you can use the bash file 'compaison.sh'.
This is used to express the deadlock situation that occurs in the peer to peer approach , but not exist in the greedy approach.
